## 1. 산술 연산자

- 이항 산술 연산자
    
    > +, -, *,/,%
    > 
- 단항 산술 연산자
    
    ++, —, +,-
    
    - [x]  +,- 단항 연산자는 부수효과가 없다
    - [ ]  +,- 단항 연산자는 부수효과가 있다
    
    ```jsx
    var x = '1'
    console.log(+x)
    console.log(x)
    
    x = true
    console.log(+x)
    console.log(x)
    ```
    
    - 문자열 연결 연산자
        
        피연산자 중 하나 이상이 문자열인 경우
        
        ```jsx
        '1' + 2; // '12'
        1 + true; // 2
        ```
        

## 2. 할당 연산자

> =, +=, -=, *=, /=, %=
> 
- 부수효과 있음
- 표현식인 문

## 3. 비교 연산자

- 동등비교, 일치 비교, 부동등 비교, 불일치 비교
    
    > ==, ===,!=, !==
    > 
    - [x]  False
    - [ ]  True
    
    ```jsx
    NaN === NaN
    ```
    
- 대소 관계 비교 연산자
    
    > < , >, <=, >=
    >

## 5. 논리 연산자

- 논리합, 논리곱, 부정
    
    > ||, &&, !
    > 
    
    ```jsx
    !0; // true
    !'Hello' //false
    'A' && 'B'; // 'B' (단축 평가)
    ```
    

## 6. 쉼표 연산자

```jsx

x = 1, y = 2, z = 3; // 3
```

## 8. typeof 연산자

피연사자의 데이터 타입을 문자열로 반환

```jsx
typeof NaN // "number"
typeof undefined // "undefined"
typeof null // object
```

- 주의

```jsx
var foo = null;
typeof foo === null; // false
foo === null; // true
```

## 9. 지수 연산자

```jsx

```
